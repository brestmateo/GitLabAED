El propósito es gestionar de manera eficiente la inscripción de estudiantes a diversos cursos o talleres, permitiendo administrar el cupo disponible y obtener estadísticas sobre la demanda académica.
El sistema permite realizar las siguientes acciones:
Gestión de Cursos: Listado de la oferta académica con control de cupos totales e inscriptos.
Registro de Estudiantes: Alta de nuevos alumnos en el sistema.
Inscripción: Asignación de estudiantes a cursos, validando automáticamente la disponibilidad de cupos.
Estadísticas: Reporte detallado de inscriptos por curso e identificación del curso con mayor demanda.
La solución fue implementada íntegramente en Python, utilizando archivos de texto plano (.txt) como mecanismo de persistencia de datos, cumpliendo con los requisitos de modularización, manejo de errores y validaciones de la cátedra.

Instrucciones de uso:
Clone este repositorio o descargue el archivo main.py.
Abra una terminal o consola en la carpeta donde se encuentra el archivo main.py.
Ejecute el programa con el siguiente comando:
"python main.py"
Siga las instrucciones que aparecerán en el menú de la consola para interactuar con el sistema. Los archivos de datos (cursos.txt, estudiantes.txt, inscripciones.txt) se crearán automáticamente al ejecutar el programa por primera vez si no existen.
